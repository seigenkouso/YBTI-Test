本测试基于 **[ヨルシカ (Yorushika) ](https://yorushika.com/)** 音乐作品与 MBTI 16 型人格理论构建的非官方同人项目。

---

### 维度解析

本项目基于 n-buna 作品中常见意象，将 MBTI 维度进行了二次解构：

- **「表达」云 (Cloud) vs 海 (Sea)**：向外散发的自我表达 vs 向内沉淀的独自承受。
- **「感知」夏 (Summer) vs 夜 (Night)**：执着于具象的现实细节 vs 游离于抽象的哲学虚无。
- **「应对」辞 (Words) vs 音 (Music)**：试图用逻辑解构痛苦 vs 依靠直觉与旋律共鸣。
- **「步调」步 (Walk) vs 待 (Wait)**：具有明确目的地的寻找 vs 停留在原地的随遇而安。
